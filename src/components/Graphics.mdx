# Graphics

## Props

http://pixijs.download/dev/docs/PIXI.Graphics.html

### draw

Draw graphics inline using the imperative `PIXI.Graphics` API.

> Use with caution, the `draw` method is costly. If the component needs to update frequently
> it might be better to use a cached texture instead.
>
> Make sure to call `g.clear()` if you'd like to redraw the graphics.

## Usage
<Playground>
  {() => {
    const draw = React.useCallback(g => {
      // clear the graphics
      g.clear()

      // start drawing
      g.beginFill(0xff3300)
      g.lineStyle(4, 0xffd900, 1)

      g.moveTo(50, 50)
      g.lineTo(250, 50)
      g.lineTo(100, 100)
      g.lineTo(50, 50)
      g.endFill()

      g.lineStyle(2, 0x0000ff, 1)
      g.beginFill(0xff700b, 1)
      g.drawRect(50, 150, 120, 120)

      g.lineStyle(2, 0xff00ff, 1)
      g.beginFill(0xff00bb, 0.25)
      g.drawRoundedRect(150, 100, 300, 100, 15)
      g.endFill()

      g.lineStyle(0)
      g.beginFill(0xffff0b, 0.5)
      g.drawCircle(470, 90, 60)
      g.endFill()
    }, []);

    return (
      <Stage width={600} height={300} options={{ backgroundColor: 0xffffff }}>
        <Graphics
          draw={draw}
        />
      </Stage>
    );
  }}
</Playground>
