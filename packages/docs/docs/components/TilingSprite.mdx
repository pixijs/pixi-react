# TilingSprite

## Props

https://pixijs.download/dev/docs/PIXI.TilingSprite.html

## Usage

```jsx live
  <Stage width={500} height={300}>
    <TilingSprite image={'https://s3-us-west-2.amazonaws.com/s.cdpn.io/693612/p2.jpeg'}
                  width={500}
                  height={300}
                  tilePosition={{ x: 100, y: 150 }}
                  tileScale={{ x: 0.1, y: 2 }} />
  </Stage>
```
