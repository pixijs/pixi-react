# SimpleMesh

## Props

https://pixijs.download/dev/docs/PIXI.SimpleMesh.html

## Usage

```jsx live noInline
const { uvs, vertices, indices } = makeSimpleMeshData();

render(
<Stage width={500} height={300} options={{ backgroundColor: 0xeef1f5}}>
<SimpleMesh image="https://s3-us-west-2.amazonaws.com/s.cdpn.io/693612/placeholder.png"
      uvs={uvs}
      vertices={vertices}
      indices={indices}
      drawMode={PIXI.DRAW_MODES.TRIANGLES}
/>
</Stage>
);
```
