# Sprite

## Props

https://pixijs.download/dev/docs/PIXI.Sprite.html

## Usage

```jsx live
<Stage width={300} height={300} options={{ backgroundColor: 0xeef1f5 }}>
  <Sprite
    image="/img/coin.png"
    scale={{ x: 0.5, y: 0.5 }}
    anchor={0.5}
    x={150}
    y={150}
  />
</Stage>
```
